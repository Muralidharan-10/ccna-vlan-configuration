This project demonstrates VLAN configuration on Cisco switches to logically separate networks and enable secure communication.

## Tools Used
- Cisco Packet Tracer
- Cisco Switches
- Command Line Interface (CLI)

## Project Overview
- Created multiple VLANs
- Assigned access ports to VLANs
- Configured trunk ports
- Verified inter-VLAN communication

## Configuration Steps
1. Created VLANs on switch
2. Assigned switch ports to VLANs
3. Configured trunk links between switches
4. Verified VLAN status using show commands
5. Tested connectivity using ping

## Outcome
Successful communication between devices within the same VLAN and across VLANs.
